Universal referral code program and marketplace.

== Description ==

[RefRef](https://refref.app) is a consumer-focused referral program for consumers to earn cash from their referrals and marketplace that features growing businesses and brands.

* Consumers only need to remember ONE referral code that can be used across multiple merchants. With referring made easier, who wouldn't!
* A consumer-facing marketplace that increases your brand awareness to much more consumers.
* Simple integration. One click. That's all it takes.

= Universal Referral Code =

* RefRef consumers create their own personal, unique referral code and share it with their friends.
* The same referral codes can be applied across all RefRef's partnering brands in the marketplace.
* Universal referral program allows for cross-exploration of merchants, increasing brand awareness and sales for all partnering merchants.

= RefRef Marketplace =

* Features RefRef partner brands in a consolidated space

= Merchant Portal =

* Full control over rewards approval
* Fraudulent referrals automatically flagged up
* Customize your referral reward offerings
* Track your referral earnings

== Installation ==

**Requires WooCommerce** 2.3.3 or higher

= From your WordPress dashboard (recommended) =

1. Go to **Plugins > Add New**
2. Search for: `RefRef`
3. Click on **Install Now** to install and activate the plugin

= From WordPress.org =

1. Download the latest version zip file
3. Go to **Plugins > Add New**
4. Click on **Upload Plugin** to upload the zip file
5. Activate the plugin after the installation

= Configure =

1. Go to **WooCommerce > Settings > Integration > RefRef**
2. Fill-out your `Merchant ID` and `Token`. You can get them from [https://merchant.refref.app/connect](https://merchant.refref.app/connect))
3. Click on **Save changes** and it's done!

**Please make sure that the plugin can make outbound requests for the API calls
